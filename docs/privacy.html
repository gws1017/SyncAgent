Text RPG 개인정보처리방침

최종 수정일: 2026-07-10

본 방침은 "Text RPG" 애플리케이션(이하 "앱")이 수집·이용하는 정보와 그 목적을 설명합니다. 이 앱은 텍스트 기반 방치형(아이들) RPG 게임입니다.

1. 수집하는 정보

앱은 다음의 정보만 수집합니다.

  • 게임 진행 데이터 (레벨, 골드, 장비, 업그레이드 등) — 기기 내부 저장소에 로컬로 저장됩니다.
  • 클라우드 동기화(선택 기능) — 사용자가 "동기화 코드"를 생성하여 PC/모바일 간 세이브를 연결하는 경우에 한해, 위 게임 진행 데이터가 Google Firebase(Realtime Database)로 전송·저장됩니다. 이 기능은 사용자가 직접 켜야만 동작하며, 기본값은 꺼짐입니다.
  • Firebase 익명 인증 ID — 클라우드 동기화 사용 시, 기기를 식별하기 위한 임의의 익명 ID가 발급됩니다. 이름, 이메일, 전화번호 등 개인 식별 정보는 수집하지 않습니다.
  • 광고 관련 정보 — 앱에 광고가 포함된 경우, 광고 네트워크 (예: Google AdMob)가 광고 제공·측정·부정 방지 목적으로 광고 ID(AAID), 기기·앱 정보, IP 주소 등을 수집·처리할 수 있습니다. 이 처리는 해당 광고 사업자의 정책에 따릅니다.

2. 광고

앱은 무료로 제공됩니다. 유료 결제 대신, 선택적인 편의 기능 (예: 자동 합성, 가방 확장 등)을 이용하기 위해 제3자 광고(주로 보상형 광고, 필요 시 배너·전면 광고)를 시청하도록 할 수 있습니다. 광고 표시를 위해 Google AdMob 등 광고 SDK가 사용될 수 있으며, 해당 SDK는 위에서 설명한 광고 관련 정보를 처리할 수 있습니다.

광고 시청은 필수가 아니며, 핵심 게임 진행(전투·성장)은 광고 없이도 가능합니다. 광고 개인화는 기기 설정(예: Android의 광고 개인 최적화 해제)으로 제한할 수 있습니다. 광고 사업자의 자세한 처리 내용은 각 사업의 개인정보처리방침을 참고해 주세요.

3. 수집하지 않는 정보

앱은 이름, 이메일, 전화번호 등 직접적인 개인 식별 정보를 요구하지 않습니다. 연락처, 위치, 사진, 카메라, 마이크 등에 접근하지 않습니다.

4. 알림 권한

앱은 게임 진행 알림(레벨업 등)과 백그라운드 실행 유지를 위한 상시 알림을 표시하기 위해 알림 권한을 사용합니다. 알림 내용은 게임 진행 정보이며, 다른 목적으로 사용되지 않습니다.

5. "프라이버시 모드" 기능에 대하여

앱 내 설정에는 "프라이버시 모드"라는 선택적 기능이 있습니다. 이 기능을 켜면 앱의 창 제목, 알림 문구, 상태 표시줄 아이콘 이름이 "sync agent"라는 일반적인 명칭으로 바뀝니다 (예: 공공장소에서 화면을 켜놨을 때 게임 화면이라는 걸 덜 눈에 띄게 하기 위함). 이 기능은 표시되는 텍스트/아이콘만 바꿀 뿐이며, 앱의 실제 기능, 데이터 처리 방식, 패키지 정보를 변경하거나 숨기지 않습니다. 기본값은 항상 꺼짐 상태입니다.

6. 데이터 보관 및 삭제

클라우드에 업로드된 세이브 데이터는 사용자가 생성한 동기화 코드에 연결되어 보관됩니다. 데이터 삭제를 원하시면 데이터 삭제 요청 페이지의 안내에 따라 요청해 주세요.

# SyncAgent

겉보기엔 백그라운드 동기화 에이전트, 실은 트레이에 숨어 도는 방치형(idle) RPG입니다.

작업표시줄 트레이에 조용히 떠 있다가, 더블클릭하면 대시보드가 열려서 캐릭터 성장 현황을
확인하고 업그레이드/특성/장비에 투자할 수 있습니다. 평소엔 알림창도 거의 안 띄우고
`[sync] ...` 형식의 로그성 문구만 남기기 때문에, 화면을 슬쩍 봐도 진짜 동기화 툴처럼 보입니다.

## 화면 구성

트레이 아이콘을 더블클릭(또는 우클릭 → 상태 대시보드)하면 작은 창이 뜹니다.

| 탭 | 내용 |
|---|---|
| 현황 | 레벨/XP, 체력, 골드, 최근 이벤트, 프레스티지 |
| 업그레이드 | 골드로 사는 영구 강화 (XP/골드/드랍률/공격력) |
| 특성 | 레벨업 시 받는 포인트로 투자하는 클래스 전용 특성 3종 |
| 던전 | 현재 스테이지, 적 HP/방어력/공격력, 내 공격력/받는 피해 |
| 장비 | 드랍된 장비 장착/해제, 같은 등급 3개 합성 |

## 게임 시스템

- **직업 3종** — 전사(탱커) / 마법사(폭딜+자가회복) / 도적(회피+다회타). 한 번 고르면 못 바꿉니다.
- **던전** — 매 틱 자동으로 적을 공격하고, 적도 반격합니다. 공격력이 적 방어력을 못 넘기면
  데미지가 0이 되어 진행이 멈추고, 체력이 0이 되면 전투가 리셋됩니다. 업그레이드/장비/특성
  투자가 없으면 반드시 어딘가에서 막히도록 설계했습니다.
- **장비** — 일반/희귀/영웅/전설 4등급, 공격력/XP/골드/드랍률/방어력/체력흡수/공격속도 중
  랜덤 스탯. 같은 등급 3개를 합성하면 확률적으로 상위 등급이 나옵니다.
- **특성** — 레벨업마다 포인트 1개, 평생 누적 15포인트 한도. 클래스별로 3가지 특성이 있고
  다 찍을 수는 없게 막아놨기 때문에 매번 골라서 투자해야 합니다.
- **프레스티지** — 일정 스테이지 이상 도달하면 레벨/장비/스테이지를 초기화하고 영구 보너스를
  받습니다. 다음 회차 요구 스테이지는 매번 더 높아집니다.

## 빌드

Windows + Visual Studio 2022 (MSVC) + CMake 필요.

```powershell
cmake -S . -B build -A x64
cmake --build build --config Release
```

빌드 결과물은 `build/Release/syncagent.exe` 입니다.

## 폴더 구조

```
src/
  game.h / game.cpp        게임 로직 (플랫폼 비종속)
  equipment.h / equipment.cpp  장비/합성 시스템 (플랫폼 비종속)
  platform.h / platform_win.cpp  세이브 파일 경로 등 OS 종속 부분
  dashboard.cpp             ImGui + DirectX11 대시보드 UI
  tray.cpp                  Win32 트레이 아이콘 / 알림
  main.cpp                  진입점, 메시지 루프
  app.ico / app.rc          앱 아이콘 리소스
tools/
  make_icon.ps1             트레이/앱 아이콘 생성 스크립트
```

게임 로직(`game.cpp`, `equipment.cpp`)은 Windows API를 쓰지 않아서, 나중에 다른 플랫폼으로
포팅할 때도 그대로 재사용할 수 있도록 분리해뒀습니다.

# 설계 백로그

당장 이번 버전엔 안 넣고, 다음 버전에서 검토할 아이디어들.

## 2차 특성에 디버프 도입 (제안: 2026-07-24)

지금은 2차 특성이 1차와 매칭 스탯 기준으로 동일한 계수를 쓴다(1.1.0에서 조정함).
다음 단계로, 2차 특성에 **트레이드오프(디버프)를 넣어서 "그냥 더 센 버전"이 아니라
"방향성이 있는 선택"이 되게 하자는 제안.

예시 (전사 기준):
- 광전사: 공격력 +4% / 받는 피해 +2%
- 수호자: 방어력 +2% / 공격속도 -1%

핵심 규칙: **디버프가 있으니 버프 폭은 1차보다 크게** — 순수 상위호환이 아니라
"하이리스크 하이리턴" 포지션으로 차별화.

### 확인/논의 필요한 점
- 디버프 수치를 클래스별로 다르게 줄지, 공통 비율로 통일할지
- 특성 UI(TabTalent)에 디버프도 같이 표시해야 함 — 지금은 버프 한 줄만 보여줌
- 밸런스 수치는 실제 적용 전에 시뮬레이션/재계산 필요 (1.1.0 밸런스 패치가
  실전에서 어떻게 자리잡는지 보고 나서 진행하는 게 안전)

## 경험치 소요시간 단축 - 보상형 광고 (제안: 2026-07-24)

XP 곡선(1.1.0에서 1.30→1.25로 완화)과 별개로, 유저가 원할 때 소요시간을
단축할 수 있는 선택적 광고 보상 기능. 스토어 정책/개인정보처리방침에 이미
광고 관련 문구는 반영해둔 상태(docs/privacy.html).

## 콘텐츠 확장 필요성 (제안: 2026-07-24)

진행 속도가 빨라지면(XP 곡선 완화 등) "게임을 다 했다"는 느낌이 더 빨리 올 수
있음 — 프레스티지 이후의 목표/콘텐츠를 늘리는 방향도 같이 고민 필요
(예: 추가 스테이지 구간, 새 장비 등급, 클래스별 추가 특성 슬롯 등).

Text RPG — 데이터 삭제 요청

앱 이름: Text RPG  |  최종 수정일: 2026-07-13

이 페이지는 Google Play에 등록된 앱 Text RPG의 클라우드 저장 데이터 삭제를 요청하는 방법을 안내합니다.

삭제 요청 방법

  1. 사용 중인 동기화 코드를 확인합니다. (앱 현황 탭 → 클라우드 동기화)
  2. GitHub 저장소에 Issue를 엽니다:
    github.com/gws1017/SyncAgent — 새 Issue 만들기
  3. 제목에 Data deletion request 또는 데이터 삭제 요청을 적습니다.
  4. 본문에 삭제할 동기화 코드를 적어 제출합니다. (다른 개인정보는 적지 않아도 됩니다.)
  5. 요청 확인 후 30일 이내에 해당 클라우드 데이터를 삭제합니다.

삭제되는 데이터

  • Firebase에 저장된 게임 진행 세이브 (레벨, 골드, 장비, 업그레이드 등)
  • 해당 동기화 코드에 연결된 클라우드 동기화 기록
  • 동기화에 사용된 Firebase 익명 인증 ID (해당 코드 경로에 묶인 범위)

삭제되지 않거나 기기에 남는 데이터

  • 기기 로컬 세이브 — 폰/PC에만 있는 진행 데이터는 서버에 없으므로 이 요청으로 지워지지 않습니다. 앱 삭제 또는 앱 데이터 삭제로 제거할 수 있습니다.
  • 광고 네트워크(향후 AdMob 등)가 기기에 저장하는 광고 ID·측정 데이터는 해당 사업자 정책에 따르며, 기기 설정에서 광고 개인화를 제한할 수 있습니다.

보관 기간

  • 클라우드 세이브: 삭제 요청이 없으면 동기화 코드가 유지되는 동안 보관됩니다.
  • 삭제 요청 접수 후: 보통 30일 이내 삭제하며, 백업/로그에 남은 경우 최대 90일 이내에 완전 제거를 목표로 합니다.
